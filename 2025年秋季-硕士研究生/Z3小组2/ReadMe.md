## 思考

如我们在最后的讨论中所说，目前的大语言模型在论证证明等语言输出领域是十分有效的，但对于一些简单的加减乘除运算却表现较差，这反映了目前的大语言模型与真正的类脑智能的核心差距，如何在大语言模型中插入一个专门负责数理逻辑的模块是一个很有意义的研究方向
